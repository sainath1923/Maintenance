# Building Maintenance Management System (MERN)

MERN-based Building Maintenance Management System for Dubai properties with Tenant, Supervisor, and Admin modules.

## Structure
- backend: Node.js/Express REST API with MongoDB
- tenant: React SPA for Tenant module
- supervisor: React SPA for Supervisor module
- admin: React SPA for Admin module

## Scripts
- npm run server
- npm run tenant
- npm run supervisor
- npm run admin
- npm run dev

Fill in server and client implementations according to the project spec.
